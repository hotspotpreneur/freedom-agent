# HealthStoreUK — Quick Setup Guide

## Phase 1: WordPress Setup (When DNS Resolves)

### 1. Install WordPress
- Log into GoDaddy → Hosting → your plan
- Use "One-Click Install" or navigate to your domain's folder
- Install WordPress (free, latest version)
- Note your admin credentials

### 2. Install Essential Plugins
Go to **WordPress Dashboard → Plugins → Add New**:

Search and install (free versions):
- **WooCommerce** — for product listings
- **Elementor** — page builder
- **Rank Math SEO** — for SEO
- **Astra** — theme (can install from Appearance → Themes)

**Speed & Security:**
- **WP Rocket** — (paid, ~$49/yr) — worth it for speed
- **Wordfence** — free security plugin
- **Autoptimize** — (free) — combine with WP Rocket for extra speed

**For Affiliate Links:**
- **ThirstyAffiliates** or **Pretty Links** — cloaks affiliate links

### 3. Activate & Configure Theme
- Go to **Appearance → Themes**
- Activate **Astra**
- Go to **Appearance → Astra Options → Starter Templates**
- Import a WooCommerce template (search "shop" or "store")

### 4. Configure WooCommerce
- **WooCommerce → Setup Wizard**
- Skip the "Add Products" for now — we'll add via affiliates later
- Set up: Currency (GBP), Measurements (UK)

### 5. Configure SEO
- **Rank Math → Setup Wizard**
- Follow the prompts
- Set your focus keyword: "health products", "supplements", etc.

## Phase 2: Content & Affiliate Strategy

### Initial Pages to Create
1. **Home** — Hero banner, featured products, categories
2. **Shop** — Product listings (affiliate links)
3. **About Us** — Story about healthstoreuk.com
4. **Blog** — Product reviews, health guides
5. **Contact** — Simple contact form

### Content Strategy
- Product comparison posts (e.g., "Best Vitamin D Supplements 2026")
- Buying guides
- "Best of" roundups

### Affiliate Programs to Join
- **Amazon Associates** — Universal, low commissions but volume
- **Health supplement niche programs** — Higher commissions:
  - Holland & Barrett affiliate program
  - Vitamin Planet
  - MyProtein (for sports/health)
- **ShareASale** — Health/niche affiliates
